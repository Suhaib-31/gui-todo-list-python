# GUI To-Do List Application 📋

## Overview

A desktop To-Do List application developed using Python and Tkinter. The application allows users to manage daily tasks with a graphical user interface and file-based task storage.

---

## Features

- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Save Tasks Automatically
- Persistent File Storage
- User-Friendly GUI

---

## Technologies Used

- Python 3
- Tkinter
- File Handling

---

## How to Run

```bash
python main.py
```
